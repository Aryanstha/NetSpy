### Dependencies

**`NetSpy`** requires the following programs to run properly - 
- `php`
- `Go`
- `git`

### Installation On Kali Linux 


```bash
$ git clone https://github.com/aryanstha/NetSpy
$ cd NetSpy
$ go run main.go
```
